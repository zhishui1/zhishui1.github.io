CSV文件

csv全称’Comma-Separated Values‘，每行对应电子表格中的一行，单元格之间用逗号分隔。

  • 值没有类型，全为字符串
  • 无多个工作表
  • 不能嵌入图像或图表
  • 无字体大小和颜色设置
  • 无法操作单元格，宽高合并等
